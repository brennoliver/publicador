# Formatos Suportados — Instagram Publisher

## Imagem (Feed)

- **Extensões aceitas**: JPG, JPEG
- **Tamanho máximo**: 8MB
- **Dimensões recomendadas**: 1080×1080px (quadrado), 1080×1350px (retrato), 1080×566px (paisagem)
- **Aspect ratio**: entre 4:5 e 1.91:1
- **Número de arquivos**: 1

## Carrossel

- **Extensões aceitas**: JPG, JPEG (todas as imagens devem ser do mesmo tipo)
- **Número de slides**: mínimo 2, máximo 10
- **Tamanho máximo por imagem**: 8MB
- **Dimensões**: todas as imagens devem ter o mesmo aspect ratio
- **Recomendado**: 1080×1080px para consistência entre slides

## Reels (Vídeo)

- **Extensões aceitas**: MP4, MOV
- **Tamanho máximo**: 1GB
- **Duração**: 3 segundos a 90 segundos
- **Resolução mínima**: 720p
- **Aspect ratio recomendado**: 9:16 (vertical)
- **Frame rate**: 23 a 60 fps

## Stories

- **Extensões aceitas imagens**: JPG, JPEG
- **Extensões aceitas vídeos**: MP4, MOV
- **Dimensão recomendada**: 1080×1920px (9:16 vertical)
- **Duração máxima (vídeo)**: 60 segundos
- **Tamanho máximo**: 4GB (vídeo), 8MB (imagem)

## Legenda

- **Máximo de caracteres**: 2200
- **Máximo de hashtags**: 30
- **Primeira quebra de linha**: o Instagram trunca após 3 linhas — colocar o mais importante primeiro

## Limites da API

- **Posts por 24h**: máximo 25 publicações via API por conta
- **Rate limit**: respeitar delays de 30 segundos entre criação e publicação do container
